Popover Component - Enhanced with Popper.js

  • Smart Auto-Positioning: Popovers automatically flip to stay in viewport
  • No More Cutoff: Intelligent edge detection prevents truncation
  • Auto-Adjusting Arrows: Arrow pointer follows the trigger element
  • Scroll Detection: Popovers close on scroll (prevents jarring flips)

Try it: Open a popover near the edge of the window - it automatically adjusts.

Quick Start - Copy & Paste This:

<!-- Add Popper.js to your <head> -->
<script src="https://cdn.jsdelivr.net/npm/@popperjs/core@2.11.8/dist/umd/popper.min.js"></script>

<!-- Basic popover structure -->
<button class="ptn-popover-trigger" 
        data-ptn-popover-trigger="popover1" 
        aria-expanded="false"
        aria-controls="popover1">
    Show Info
</button>

<div id="popover1" 
     class="ptn-popover" 
     role="dialog" 
     aria-labelledby="popover1-title"
     data-backdrop="false">
    <div class="ptn-popover-arrow" data-popper-arrow></div>
    <h3 id="popover1-title" class="ptn-popover-header">Title</h3>
    <div class="ptn-popover-content">
        <p>Content goes here</p>
    </div>
</div>

That's it. The JavaScript handles all positioning automatically using Popper.js.

Usage Notes

What's Required:

  • Popper.js: Include the script in your <head> tag
  • Arrow element: Add <div class="ptn-popover-arrow" data-popper-arrow></div> inside each popover for the pointer
  • Unique IDs: Each popover needs a unique ID that matches its trigger's data-ptn-popover-trigger attribute

Features:

  • Smart Positioning: Automatically flips popover to top/bottom/left/right based on available space
  • Edge Detection: Prevents popover from being cut off at viewport edges
  • Arrow Positioning: Arrow automatically follows the trigger element
  • Scroll Behavior: Popovers automatically close when scrolling starts (prevents jarring flips)
  • Keyboard Navigation: Press Escape to close, or click outside
  • Backdrop Control: Add data-backdrop="false" to disable overlay, or data-backdrop="true" to enable it (default is false)
  • Size Control: Add size classes: ptn-popover-small, ptn-popover-medium, ptn-popover-large, ptn-popover-wide, ptn-popover-full, or ptn-popover-auto
  • Focus Management: Automatically focuses first interactive element when opened
  • Menu Support: Arrow key navigation for role="menu" popovers

Browser Support:

  • Works in all modern browsers (Chrome, Firefox, Safari, Edge)
  • Popper.js handles the heavy lifting of positioning calculations
  • Degrades gracefully if Popper.js fails to load (basic positioning still works)

Configuration Options:

The Popper.js configuration in the code includes:

  • placement: Default is 'bottom', but auto-flips as needed
  • offset: 8px gap between trigger and popover
  • flip: Tries bottom → top → right → left to find best position
  • preventOverflow: 10px padding from viewport edges
