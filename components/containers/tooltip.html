Usage Example

<!-- Solid style (default), positioned on top -->
<div class="tooltip-container" data-position="top">
  <button class="tooltip-trigger"
          data-icon="info"
          data-style="solid"
          aria-label="More information"
          data-tooltip="Your tooltip text here">
  </button>
  <div class="tooltip-content" role="tooltip" aria-hidden="true"></div>
</div>

<!-- Outlined style, positioned on top -->
<div class="tooltip-container" data-position="top">
  <button class="tooltip-trigger"
          data-icon="info"
          data-style="outlined"
          aria-label="More information"
          data-tooltip="Your tooltip text here">
  </button>
  <div class="tooltip-content" role="tooltip" aria-hidden="true"></div>
</div>

Available icons: info, question, warning, error, success

Available styles: solid (default), outlined

Available positions: top, bottom, left, right

Usage notes

  • To be comprehensive, this example shows far more styles than you would practically use. Be selective and consistent. Don't use multiple versions if one will do. Use the info OR question icon, not both. Use all solid, or all outlined, depending on what fits your design, but don't mix both.
  • Available position values: top, bottom, left, right
  • Available icon values: info, question, warning, error, success
  • Example usage for icons:
    • Info or question: additional information, definition, short example
    • Warning or error: as a badge alerting users to a condition or element that needs attention.
    • Success icon: as a note that an element is correct
