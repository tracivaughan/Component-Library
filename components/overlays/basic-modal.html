Usage notes

  • Instructions:
    1. Include the script in the HTML
    2. Add the modal markup anywhere in the HTML. Each modal must have a unique ID.
    3. Call the launching js [openModal('myModal')] from a button (strongly preferred), link, or script.
  • To set a custom max width, set the data-modal-size attribute in the modal markup. This will let you adjust the width to match the expected content. Leaving this out will make the modal the default size, which is set in the .js file (800px). You can adjust that max-width in the script to change the default size universally.
  • Some default styling is set in the .js. You can also adjust the formatting in the CSS, including adding additional classes to visually distinguish different modals (for example, errors, warnings, confirmations could be themed).
