Usage notes

  • Instructions:
    1. Include the script in the HTML
    2. Add the modal markup anywhere in the HTML. Each modal must have a unique ID.
    3. Call the launching js [openModal('myModal')] from a button (strongly preferred), link, or script.
  • To set a custom max width, set the data-modal-size attribute in the modal markup. This will let you adjust the width to match the expected content.
  • Modal design can be set in the CSS. There is also default styling for the overlay and modal div in the JavaScript, clearly labeled.
