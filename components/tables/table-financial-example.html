Usage notes

  • Slight cell border & tighter cells than standard example.
  • TFoot part is optional.
  • Table rows stack on small screens, up to 768px.
  • Includes class to add to cells that contain negative amounts; can also highlight cell background.
  • Required caption available to screen readers but accessibly hidden from visual display.
  • Any added link/focus behaviour will inherit from existing stylesheet; make sure those :link, :hover, and :focus states are accessible.
  • Table will expand to full available width; add constraining container or max-width to control width if needed.
