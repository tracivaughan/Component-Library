Usage notes

  • Table rows stack on small screens, up to 768px.
  • Useful for including multiple actions or button-controlled actions.
  • Colors can be changed to better fit the specific context of your application.
  • Unused icons can be deleted from the svg icon definitions if desired, but leaving them will have no negative effect.
  • Includes optional status pills for conditions or statuses:
    • Positive (green) = completed, on-track; positive state
    • Negative (red) = warning, error, off-track; negative state/li>
    • Pending (blue) = pending; neutral condition
    • In-progress (yelllow) = in progress; neutral or alert condition
  • Required caption available to screen readers but accessibly hidden from visual display.
  • Link/focus behaviour will inherit from main stylesheet; make sure those :link, :hover, and :focus states are accessible.
  • Table will expand to full available width; add constraining container or max-width if needed.
